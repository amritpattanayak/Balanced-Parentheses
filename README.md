# Balanced-Parentheses
C# program to check balanced parentheses for an input string.
It includes the (), {}, [].
Stack is sused to solve the problem.

Example:
(10 + 20) = 30            //Balanced
{Balanced} {Parentheses}   //Balanced
[{loren (ipsum)}]          //Balanced
[loren (ipsum]             //Not balanced
loren) (ipsum)             //Not balanced
